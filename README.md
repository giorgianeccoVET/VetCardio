# VetCardio V1

Prima versione funzionante con:
- login via email
- proprietari e pazienti
- visite
- ecocardiografia, ECG ed ecografia addominale
- sincronizzazione Supabase

Configurazione:
1. Eseguire `schema.sql` nel SQL Editor di Supabase.
2. Inserire Project URL e anon key in `config.js`.
3. Caricare i file nel repository GitHub `VetCardio`.
4. Fare deploy con Vercel.
